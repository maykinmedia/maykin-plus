# Welkom bij {MAYKIN}+

Onder de noemer **Maykin plus** voert Maykin allerlei activiteiten uit om open source software duurzaam te maken en breder gebruik binnen de Nederlandse overheid te stimuleren.

In deze repository staan de openbare documenten die voor iedereen handvaten geven om hierbij te helpen. Bekijk alles op de [website!](https://maykinmedia.github.io/maykin-plus/).
